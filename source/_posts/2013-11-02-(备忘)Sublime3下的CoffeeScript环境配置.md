先安装PackageControl这个神器:

``` shell
cd ~/.config/sublime-text-3/Packages

git clone https://github.com/wbond/sublime_package_control.git "Package Control"
```
然后启动 sublime，按下 Ctrl + Shift + P

```
install package:  BetterCoffeeScript
```
